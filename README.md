# CS50P - Lesson 0: Functions and Variables

This is my repository for tracking my Python learning journey with Harvard University's CS50p course, taught by Professor David Malan.

## What I Learned in Lesson 0:
* **Side Effects:** Learned how running a program in the terminal using `py hello.py` provides the output side effect.
* **Arguments & Variables:** Practiced using the `input()` function to capture variables (like names, ages, and weights) and `print()` to output them.
* **String Formatting:** Tested spacing using both the `+` concatenation operator and commas `,` for automatic spacing.

## My Personal Code Analogies:
For a grounded understanding, I imagine real-world examples and code them down. In this lesson, I built custom prompts to calculate and print out a user's age and weight metrics side-by-side!
